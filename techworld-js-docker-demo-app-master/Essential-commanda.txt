# It is a javascript full fledge application in docker 
#Mongo, MongoDB, Express are the backend requirements


1. docker pull mongo:latest
2. docker pull mongo-express:latest
3. Now run the mongo-container 
        
     docker run -d \                                                             
-p 27017:27017 \
-e MONGO_INITDB_ROOT_USERNAME=mongoadmin \
-e MONGO_INITDB_ROOT_PASSWORD=password \
--net mongo-network \
--name mongodb \
mongo:latest 

       ------> Here the by default port for mongo is 27017
              Need to overwrite the username and password
              
4. Now run the mongo-express container

    docker run -d \
> -p 8081:8081 \
> -e ME_CONFIG_MONGODB_ADMINUSERNAME=mongoadmin \
> -e ME_CONFIG_MONGODB_ADMINUSERNAME =password \
> --net mongo-network \
> --name mongo-express \
> -e ME_CONFIG_MONGODB_SERVER=mongo \
> mongo-express

     ------> Here the value of the server is the container name of mongo container
     -------> The ADMINUSERNAME and ADMINPASSWORD should be same as of mongo container 
     -------> NOTE: Always remember if you are not assigning any username and password to the GUI of the mongo express by default username is admin and password is pass 


     -----> use -e ME_CONFIG_BASICAUTH_USERNAME=any username you like
                -e ME_CONFIG_BASICAUTH_PASSWORD=anypassword you like
               In the docker run command at 4th point



5. Cerate a database name user-account in the GUI of mongo-express
